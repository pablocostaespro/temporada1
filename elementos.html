Sobre o template

A seguir, podemos ver a definição dos elementos para a Espro. É fundamental que esses elementos tenham as características respeitadas. É importante que se tente, ao máximo, seguir o tamanho aproximado dos textos para cada elemento que vemos aqui - quando se extrapola em muitos caracteres ou então o trecho selecionado é muito pequeno, temos problemas com a responsividade do material por conta da desconfiguração dos padrões de medidas e layout do Webflow.

Qualquer dúvida sobre os elementos e o template estamos à disposição!
Atenciosamente, Programação Visual.
